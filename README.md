# RateGov
Saаs-based system for evaluating the work of government agencies and business institutions “RateGov”

## Technologies used:
  - ### Back-end:
    - Node.js
    - Express
    - TypeScript
    - MongoDB
    - Web-Sockets
    - REST API
    - docker-compose
    
  - ### Front-end:
    - JavaScript
    - React
    - Redux
    - Web-Sockets
    - Reducers
    - Ant-Design
